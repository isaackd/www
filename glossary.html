Glossary

CSS
Cascading Style Sheets. Used to style websites
Block Element
Takes up the entire line. An example of this is the div element
Responsive
Responsive web design changes with the size of the screen.
Attribute
An attribute changes the behaviour of an element. For example, the title attribute will add a title to an element that displays on hover.
Class
A class is used to target elements with CSS selectors.

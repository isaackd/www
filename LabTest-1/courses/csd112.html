Intro To Web Dev

HTML and CSS are the fundamental technologies for creating web interfaces. After a brief introduction to the World Wide Web, students learn the HTML elements that are used in all web pages, including page layout elements, tables, forms, and more modern media elements for video and audio. Students also learn advanced styling techniques using CSS to give web sites custom layouts and appearances, including responsive design and CSS animation. Throughout the course, accessibility standards to make web sites usable to the widest possible audience are highlighted. Students use modern web browsers, GitHub, and Visual Studio Code to create working web sites.
